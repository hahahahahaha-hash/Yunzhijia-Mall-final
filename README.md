# ☁️ 云之家在线商城 (Yunzhijia-Mall)

这是一个基于 [Vue2 全家桶 + HTML5/CSS3 + JavaScript + Webpack + Element UI] 开发的在线商城系统。

## 🌟 项目介绍
多人协作的 B2C 在线购物商城，涵盖商品展示、登录鉴权、购物车、订单结算等核心模块，支持商品搜索筛选、购物
车数据持久化，是前端工程化结合电商业务的实战项目

## 🛠️ 技术亮点
• 负责首页、登录注册、购物车等核心模块开发，基于 Element UI 搭建组件，CSS3 实现动效优化，提升页面交互体验
• 基于 Token 实现用户登录鉴权与权限控制，处理记住密码、自动登录等场景，保障用户数据安全
• 开发购物车增删改查、价格实时计算功能，通过 localStorage 实现数据持久化，完成模块间数据联动
• 实现下单流程表单验证、数据提交，配合后端完成接口对接与跨域问题解决，遵循团队编码规范参与代码评审

## 🚀 如何运行
1. 克隆项目到本地。
2. 在终端依次输入npm install和npm run serve
3. 这会编译代码并在本地运行（通常运行在 http://localhost:8080）
